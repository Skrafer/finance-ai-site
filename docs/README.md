## Кратко о проекте

Проект посвящен концепции ИИ-ассистента по финансовой грамотности. Сайт объясняет проблему управления личными финансами, показывает ключевые функции продукта и представляет команду, работавшую над проектом.

## Состав репозитория

- Основной сайт: `index.html`, `app.html`, `developers.html`.
- Стили и интерактивность: `styles.css`, `script.js`.
- Изображения и визуальные материалы: `assets/`.
- Документация: `docs/`.
- Отчетные материалы: `report/`.

# Журнал работы

## Этап 1. Выбор идеи

Команда выбрала тему ИИ-ассистента по финансовой грамотности. Были определены основные проблемы пользователей: сложность учета расходов, отсутствие прогноза бюджета и недостаток понятных рекомендаций.

## Этап 2. Планирование структуры сайта

Была выбрана структура из трех страниц:

- главная страница продукта;
- страница с внутренней логикой приложения;
- страница команды.

## Этап 3. Подготовка контента

Команда описала функции ассистента, пользовательский сценарий, блок финансовой безопасности и распределение ролей между участниками.

## Этап 4. Разработка сайта

Были созданы HTML-страницы, стили CSS и небольшой JavaScript для интерактивности. Сайт получил адаптивное оформление, визуальные материалы и карточки участников.

## Этап 5. Подготовка отчетных материалов

Была подготовлена папка `report` с отчетом по проектной практике.

## Этап 6. Оформление документации

Для удобства материалы были разнесены по разделам: описание проекта, команда, журнал работы и полезные ресурсы.

## Текущий результат

Проект содержит сайт, отчетные материалы и документацию по основным этапам работы.

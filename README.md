# ИИ-ассистент по финансовой грамотности

## О проекте

Проект представляет собой статический сайт о сервисе "ИИ-ассистент по финансовой грамотности". Идея продукта связана с помощью пользователю в управлении личными финансами: учет расходов, планирование бюджета, цели накоплений, предупреждения о рисках и понятные рекомендации.

Сайт выполнен как презентация цифрового продукта. Он показывает, какую проблему решает сервис, какие функции в нем предусмотрены, как может выглядеть внутренняя логика приложения и кто участвовал в разработке проекта.

## Цель проекта

Цель проекта - представить концепцию финансового ассистента, который помогает пользователю лучше понимать личный бюджет и принимать более осознанные финансовые решения.

Проект направлен на то, чтобы показать не просто набор экранов, а цельную идею продукта: от проблемы пользователя до возможного решения, структуры приложения и роли команды.

## Актуальность

Личные финансы часто сложно контролировать вручную. Пользователь может помнить крупные покупки, но не замечать мелкие регулярные траты, подписки, комиссии и импульсивные расходы. Из-за этого бюджет становится непрозрачным, а финансовые решения принимаются без полной картины.

Финансовый ассистент помогает собрать данные в понятный вид. Он показывает категории расходов, помогает заметить повторяющиеся платежи, предупреждает о рисках и подсказывает, на что стоит обратить внимание.

## Основные возможности продукта

- учет доходов и расходов;
- распределение операций по категориям;
- прогноз остатка бюджета;
- поиск регулярных платежей и подписок;
- постановка целей накоплений;
- советы по экономии;
- предупреждения о финансовых рисках;
- раздел финансовой безопасности;
- понятное объяснение финансовой ситуации для пользователя.

## Структура сайта

Проект состоит из трех основных страниц.

### Главная страница

Файл: `index.html`

На главной странице описаны проблема, идея продукта, основные функции и ценность финансового ассистента. Эта страница знакомит пользователя с проектом и показывает, зачем нужен такой сервис.

### Страница приложения

Файл: `app.html`

Страница описывает внутреннюю логику продукта. На ней показаны основные модули: ввод данных, анализ расходов, прогноз бюджета, цели, рекомендации и финансовая безопасность.

### Страница команды

Файл: `developers.html`

На странице команды представлены участники проекта, их роли и вклад в работу. Карточки участников можно раскрывать для просмотра подробной информации.

| Технология | Назначение |

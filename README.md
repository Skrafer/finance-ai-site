# ИИ-ассистент по финансовой грамотности

Статический сайт проектной команды о продукте "ИИ-ассистент по финансовой грамотности". Проект представляет идею мобильного финансового помощника, который помогает пользователю контролировать расходы, прогнозировать бюджет, ставить цели накоплений и замечать финансовые риски.

## Содержание репозитория

- `index.html` - главная страница с описанием проблемы, функций и ценности продукта.
- `app.html` - страница с внутренней логикой приложения, пользовательским сценарием и блоком безопасности.
- `developers.html` - страница команды, распределение ролей и вклад участников.
- `styles.css` - стили сайта, адаптивная верстка и визуальное оформление.
- `script.js` - небольшая интерактивность: появление блоков при прокрутке и раскрытие карточек участников.
- `assets/` - изображения, макеты продукта, портреты участников и QR-код.
- `report/` - отчетные материалы по проектной практике в формате Markdown и DOCX.

## Запуск проекта

Проект не требует сборки и серверной части.

1. Скачайте или клонируйте репозиторий.
2. Откройте файл `index.html` в браузере.
3. Для публикации можно использовать GitHub Pages.

## Цель проекта

Цель проекта - представить концепцию финансового ассистента, который делает управление личными финансами понятнее: показывает пользователю состояние бюджета, объясняет причины расходов и предлагает практичные рекомендации.

# Технический отчет

## Назначение документа

Документ описывает техническую часть проекта: структуру файлов, используемые технологии и особенности реализации сайта.

## Общая структура

Проект выполнен как статический сайт без серверной части. Все страницы открываются напрямую в браузере.

```text
finance-ai-site/
├── assets/
├── docs/
├── report/
├── app.html
├── developers.html
├── index.html
├── script.js
├── styles.css
└── README.md
```

## Основные страницы

| Страница | Описание |
| --- | --- |
| `index.html` | главная страница с описанием идеи продукта |
| `app.html` | страница с описанием внутренней логики приложения |
| `developers.html` | страница команды и распределения ролей |

## Стили

Файл `styles.css` отвечает за визуальное оформление сайта:

- общую цветовую схему;
- сетки и расположение блоков;
- карточки, кнопки и секции;
- адаптацию под разные размеры экрана;
- оформление страницы команды и карточек участников.

## JavaScript

Файл `script.js` используется для простой интерактивности:

- появления блоков при прокрутке страницы;
- раскрытия карточек участников команды;
- управления состоянием раскрытых карточек через `aria-expanded`.

## Изображения

Папка `assets` содержит визуальные материалы проекта: макеты интерфейса, изображения для презентации продукта, портреты участников и QR-код для страницы команды.

## Особенности реализации

Проект не использует фреймворки и внешние зависимости. Такой подход упрощает запуск: сайт можно проверить локально без установки дополнительных программ.

## Итог

Техническая часть проекта соответствует формату статического сайта. Структура файлов понятная, основные страницы разделены по смыслу, а документация вынесена в отдельные папки.

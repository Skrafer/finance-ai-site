# Итоговый отчет

## Название проекта

ИИ-ассистент по финансовой грамотности.

## Краткое описание

Проект представляет сайт о цифровом сервисе для помощи в управлении личными финансами. На сайте описаны проблема, идея продукта, основные функции, внутренняя логика приложения, безопасность и команда проекта.

## Выполненные работы

- Подготовлена структура сайта.
- Созданы страницы `index.html`, `app.html` и `developers.html`.
- Оформлены стили в `styles.css`.
- Добавлена интерактивность через `script.js`.
- Подготовлены изображения и визуальные материалы.
- Сформирована документация в папках `docs` и `report`.
- Добавлена версия отчета в формате Word.

## Полученный результат

Итогом работы стал статический сайт, который можно открыть локально или разместить на GitHub Pages. Проект выглядит как презентация идеи финансового ассистента и содержит материалы для проверки практики.

## Практическая ценность

Проект показывает, как можно представить цифровой продукт через сайт: описать проблему, объяснить пользу сервиса, показать структуру приложения и распределить вклад команды.

## Возможное развитие

В дальнейшем проект можно расширить:

- добавить реальные формы ввода доходов и расходов;
- подключить хранение данных;
- сделать расчет бюджета;
- добавить графики и аналитику;
- реализовать личный кабинет пользователя.

## Вывод

Работа над проектом позволила закрепить навыки верстки, оформления интерфейса, подготовки структуры сайта, командного распределения задач и ведения документации в репозитории.

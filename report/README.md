# Отчетная документация

Папка содержит комплект отчетных материалов по проектной практике. Документы разделены по назначению, чтобы отдельно показать описание проекта, техническую часть, вклад команды и итог работы.

## Состав отчетов

| Файл | Назначение |
| --- | --- |
| `practice-report.md` | основной отчет по проектной практике |
| `technical-report.md` | техническое описание структуры сайта |
| `team-report.md` | распределение ролей и вклад участников |
| `final-report.md` | краткий итоговый отчет по результатам работы |
| `отчет практика.docx` | версия отчета в формате Word |

## Краткая информация о проекте

Проект представляет сайт о сервисе "ИИ-ассистент по финансовой грамотности". Сайт показывает идею продукта, основные функции, внутреннюю логику приложения, блок финансовой безопасности и команду проекта.

## Быстрая навигация

- [Технический отчет](technical-report.md)
- [Командный отчет](team-report.md)

## Проверка материалов

Текстовые отчеты можно просмотреть в репозитории. Word-файл оставлен как отдельная версия отчета для передачи через личный кабинет или другую систему сдачи.

# Отчетные материалы

В папке собраны файлы, относящиеся к отчету по проектной практике.

## Состав папки

- `practice-report.md` - текстовая версия отчета для просмотра в репозитории.
- `отчет практика.docx` - файл отчета в формате Word.

Обе версии содержат материалы по одному проекту и нужны для удобной проверки работы в разных форматах.

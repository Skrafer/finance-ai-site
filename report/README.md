# Отчетные материалы

В этой папке находятся материалы по проектной практике.

## Файлы

- `practice-report.md` - основной отчет в формате Markdown для просмотра на GitHub.
- `отчет практика.docx` - версия отчета в формате Word для загрузки через личный кабинет.

Markdown-версия добавлена отдельно, чтобы документация была видна прямо в репозитории без скачивания файлов.

# Отчет по проектной практике

## Тема проекта

ИИ-ассистент по финансовой грамотности.

## Цель работы

Целью работы было подготовить сайт, который показывает идею сервиса для помощи в управлении личными финансами. На сайте нужно было описать проблему, основные функции продукта, примерную внутреннюю логику приложения и вклад участников команды.

## Актуальность

Личные финансы часто сложно контролировать без отдельного инструмента. Человек может помнить крупные траты, но при этом не замечать мелкие покупки, подписки, комиссии и повторяющиеся платежи. В итоге бюджет становится не совсем понятным: деньги уходят, но причины не всегда очевидны.

Идея проекта состоит в том, чтобы сделать финансового помощника, который собирает расходы в понятную картину, показывает слабые места бюджета и подсказывает, на что стоит обратить внимание.

## Основные задачи

- Продумать идею продукта и его пользу для пользователя.
- Подготовить структуру сайта.
- Сделать страницы с описанием продукта, приложения и команды.
- Оформить сайт так, чтобы он выглядел как законченная презентация проекта.
- Добавить раздел о финансовой безопасности.
- Подготовить отчетные материалы.

## Структура сайта

Проект состоит из трех основных страниц.

### `index.html`

Главная страница. На ней описаны проблема, идея продукта, основные возможности финансового ассистента и итоговая ценность сервиса для пользователя.

### `app.html`

Страница о работе приложения. Здесь показаны основные блоки: ввод данных, анализ расходов, прогноз бюджета, цели, рекомендации и безопасность.

### `developers.html`

Страница команды. На ней указаны участники проекта, их роли и вклад в работу.

## Возможности продукта

В проекте описаны следующие функции:

- учет доходов и расходов;
- распределение операций по категориям;
- прогноз остатка бюджета;
- поиск регулярных платежей;
- цели накоплений;
- советы по экономии;
- предупреждения о финансовых рисках;
- блок финансовой безопасности.

## Использованные технологии

- HTML - структура страниц.
- CSS - оформление и адаптивность.
- JavaScript - простая интерактивность на странице команды и анимации появления блоков.
- GitHub - хранение проекта и публикация материалов.
